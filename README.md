# Cloud Blog Platform (AWS)

## Deskripsi
Proyek ini adalah aplikasi blog berbasis cloud menggunakan AWS yang dapat diakses secara online.

## Fitur
- Login user
- CRUD artikel
- Upload gambar

- RDS
- S3
- CloudFront
- VPC
- IAM
- CloudWatch
